# Loop Engineering 大全

AIエージェントの自律化を設計・制御するための体系的アプローチ、「Loop Engineering」の全貌を解説した技術書です。

## 本書のサマリ
Loop Engineeringとは、AIエージェントに人間が毎回プロンプトを打つのではなく、*「目的発見 → 文脈取得 → 計画 → 実行 → 観測 → 検証 → 調整 → 状態保存 → 次アクション/人間へ」*という反復ループそのものを設計・運用・改善する技術です。

人間の役割は「作業者（毎ターンの指示者）」から、自動化ラインの**「設計者・承認者・監査者」**へ移ります。レバレッジの所在が*「個々のプロンプトを練ること」から「エージェントを時間をかけて統御する制御系を設計すること」*へ移動した、というのが本書の主張です。

「自律化すれば人間レビューが不要になる」は誤解です。「完了」は*主張*であって*証明*ではありません。本書では、検証・権限・状態・停止条件・人間ゲートの設計を軸に、安全かつ高効率なAIシステムを構築する方法を詳説します。
